¿Que datos se pueden editar?

No se pueden modificar los datos que Reserhab utiliza para verificar tu identidad. Puedes modificar tus datos personales y de contacto, pero podríamos pedirte que verifiques tu identidad la próxima vez que hagas una reservación o crees una habitación.

¿Qué información se comparte con los demás?

Reserhab solo proporciona los datos de contacto de los reservadores y los propietarios una vez que la reservación se haya confirmado.
